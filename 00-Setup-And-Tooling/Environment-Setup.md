# Environment Setup (macOS)

## Required Tools

- .NET SDK 8.x
- Git
- A code editor (Visual Studio Code / Rider / Visual Studio)
- Terminal access

## Verifying .NET Installation

```bash
dotnet --version
```
Expected output:

Version 8.x.x

## Why .NET SDK Matters

The SDK includes:

- C# compiler

- Runtime

- CLI tooling

All examples in this repository target:

- .NET 8

- Modern C# features

## OS Assumptions

- cmacOS is used for development

- Commands are POSIX-compatible
